# Minimalist modal text editor
    N - navigation mode
    I - insert mode

    h - move cursor left
    j - move cursor down
    k - move cursor up
    l - move cursor right
    d - delete current line
    i - go to insert mode

    Ctrl-u move cursor few lines up
    Ctrl-d move cursor few lines down
    Ctrl-w write changes to current file
    Ctrl-q quit editor

# Usage
python3 e.py e.py
